# Sillaka54

This is my first custom keyboard built with the following initial setup:
- Switches: Outemu Silent Honey Peach V3
- Keycaps: Frog Green
- MCU: RP2040 Zero
- Creator: [Squalius-cephalus](https://github.com/Squalius-cephalus)


## How to flash the keyboard firmware
Follow the instructions at [Firmware Update](https://github.com/Squalius-cephalus/silakka54/wiki/Firmware-Update).


## How to load configuration
Download [Vial](https://get.vial.today/) and follow the user manual to learn about it.

Then, load the `setup.via` to overwrite the keyboard configuration.
